## Backend

Inicie o mongoDB
```bash
  $ mongod
```

Entre na pasta `backend` do projeto e instale as dependencias
```bash
  $ yarn
  // ou
  $ npm i
```

Executando o backend
```bash
  $ yarn run dev
  // ou
  $ npm run dev
```

## Frontend

Entre na pasta `frontend` do projeto e instale as dependencias
```bash
  $ yarn
  // ou
  $ npm i
```

Executando o frontend
```bash
  $ yarn run dev
  // ou
  $ npm run dev
```
